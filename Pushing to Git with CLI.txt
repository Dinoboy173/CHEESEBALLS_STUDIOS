How to push to Github using Command Line Interface

1. Pull any contributions ahead of your computer

2. git add .
	This stages all changed files for commit
	
3. git commit -m "<your commit message here>"
	Commits the changes added ready to be pushed. Make sure you have a useful commit message that easily tells people what has been added.
	
4. git push
	Pushes all changes to the repository. Don't close Command Prompt until it has finished pushing to git.
